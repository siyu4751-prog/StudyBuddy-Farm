# StudyBuddy-Farm
# FocusPet 学习宠物养成器

一个适合学生使用的轻量级学习管理网页项目。用户可以添加学习任务、开启番茄钟、完成任务获得能量，并用能量值解锁不同的学习宠物状态。

# 项目亮点

使用 HTML、CSS、JavaScript 原生开发，不依赖框架
支持任务添加、完成、删除
支持番茄专注计时器,支持能量值、等级、累计学习数据统计,使用 localStorage 本地存储，刷新页面后数据不会丢失,页面风格清新，适合部署到 GitHub Pages 展示

# 文件结构

```text
focuspet-github/
├── index.html      # 页面结构
├── style.css       # 页面样式
├── script.js       # 交互逻辑
└── README.md       # 项目说明
```

# 使用方法

1. 下载项目文件。
2. 直接双击 `index.html` 即可在浏览器中运行。
3. 也可以上传到 GitHub 仓库，并开启 GitHub Pages 进行在线展示。

# 适合写进简历的项目描述

FocusPet 是一个基于原生 HTML、CSS、JavaScript 实现的学习效率管理工具，围绕学生日常学习场景，设计了任务清单、番茄专注计时、能量奖励、宠物养成和本地数据存储等功能。项目通过 localStorage 实现数据持久化，通过 DOM 操作实现任务动态渲染和状态更新，页面适配 PC 与移动端，适合作为前端基础能力展示项目。
